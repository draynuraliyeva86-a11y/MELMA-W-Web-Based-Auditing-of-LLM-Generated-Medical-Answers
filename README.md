# MELMA-W: Web Based Auditing of LLM Generated Medical Answers
MELMA-W is a web-based tool designed to evaluate medical answers generated by large language models (LLMs). It is the online implementation of the MELMA framework and is based on the 30-item MELMA-Q questionnaire. Users can enter a clinical question, the LLM-generated answer, and the model name, and MELMA-W evaluates the response using a structured scoring system.

The tool provides item-level scores, domain-level scores, an explanation of the scoring rationale, a radar/spider visualization, and a downloadable evaluation report. MELMA-W helps make LLM answer evaluation more standardized, reproducible, and transparent, especially for assessing medical accuracy, clinical reasoning, safety, clarity, usefulness, and overall answer behavior.
